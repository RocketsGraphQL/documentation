# Introduction
 
Welcome to Rocketgraph! This is a basic docs. You can use it as a starting point for your own project :)

## What is Rocketgraph?

A **simple**, **powerful** and **flexible** site generation tool with everything you love. You can create Authentication with email, social and OTP in minutes and authorise fine-grained permission rules on you favourite PostgresDB. You will also get your data in real-time in GraphQL API with SSL enabled right out of the box. In addition you will get serverless compute space for your own backend routes and Stripe integration so you can use your stripe customers as authenticated users.
